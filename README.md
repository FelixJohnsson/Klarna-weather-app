# Klarna Weather App
A one page React Native application for weather information. 
Using the https://www.visualcrossing.com/ Weather API.

## Getting Started
Developed using Expo.

### Running
`cd KlarnaWeatherApp`

`npm start` - Opens up expo.

I used Expo Go for testing locally.
